## 板块 5 · 想法(进行中)

**定位**:系统的灵魂——唯一"你产出"的东西。看截图产生的思考,汇聚成流,长期成为"你怎么思考"的镜子。

**构思(完整愿景,分批做)**:
1. 可编辑 ✅
2. 能打主题(按主题看想法、看想法随时间怎么变)✅
3. 好想法「精选」入脑(进 core.knowledge)✅
4. 对一条想法「深化」——问 AI 反驳/延展 ⏳(属"追问"板块,后做,别串)
5. 想法里标记重点词/句(高亮)⏳(锦上添花,低优先)
6. 远期:AI 通读所有想法,分析你的思维模式/盲区 ⏳

### 已实现(2026-07-04)
- **数据**:`core.entries` 加 `theme`(主题)、`promoted_at`(已精选);想法带 `source_item_id`(来源截图)。
- **编辑**:Ideas 页每条可改文字(`PATCH /entries/{id}`)。
- **打主题**:每条想法一排主题 chip,点一下打/取消;顶部按主题筛选(`PATCH` theme)。
- **精选入脑**:`POST /entries/{id}/promote` → body 存进 `core.knowledge`、挂主题标签、标 `promoted_at`;防重复(409)。前端「精选」按钮。
- **来源**:想法卡显示来源截图缩略,点进原图详情。
- 已 E2E 验证:编辑 + 打主题 + 精选(建 knowledge + tag + promoted)+ 防重复。

### 待做(想法板块内,以后)
- 深化(问 AI)——等「追问」板块。
- 高亮重点——想法多起来后。
- 思维模式分析——远期。
